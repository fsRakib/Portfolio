## ðŸš€ Getting Started

### Prerequisites

- A modern web browser
- Basic understanding of HTML, CSS, and JavaScript (for customization)

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/fsRakib/Portfolio.git
   ```

2. Navigate to the project directory

   ```bash
   cd Portfolio
   ```

3. Open `index.html` in your browser or use a local server

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve
   ```

4. Visit `http://localhost:8000` in your browser

## ðŸ“ Customization

### Update Personal Information

1. Edit `components/home.html` - Update name, title, and bio
2. Edit `components/about.html` - Update personal details and timeline
3. Edit `components/skills.html` - Add/remove your skills
4. Edit `components/portfolio.html` - Add your projects
5. Edit `components/contact.html` - Update contact information

### Change Colors

1. Go to the style switcher (gear icon on the left)
2. Click on any color circle to change the theme
3. Or modify the CSS variables in `css/skins/` folder

### Add New Components

1. Create a new HTML file in `components/` folder
2. Add the component to `js/component-loader.js`
3. Update navigation in `components/sidebar.html`
